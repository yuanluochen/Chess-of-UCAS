
# 五子棋游戏 - C++实现

UCAS C++课程作业，实现五子棋游戏与人机对战功能。

## 功能特性

### 基础功能
- 15×15标准棋盘
- 双人对战模式
- 人机对战模式
- 实时棋盘显示
- 胜负判定系统
- 输入合法性验证

### 人工智能模块
- 决策树算法实现
- Alpha-Beta剪枝优化
- 棋盘状态评估函数
- 智能落子策略

## 核心模块设计

### 1. 棋盘表示
- **数据结构**: 一维数组存储15×15棋盘
- **棋子编码**: 
  - 黑棋 (BLACK): 1
  - 白棋 (WRITE): -1
  - 空位 (VOID): 0
- **坐标系统**: 行(0-14) + 列(a-o)

### 2. 游戏逻辑类 (chessGame)
- **棋盘管理**: 初始化、显示、状态查询
- **游戏流程**: 轮流落子、胜负判定、平局判断
- **历史记录**: 队列存储已落棋子位置
- **游戏模式**: 支持双人对战与人机对战

### 3. 胜负判定系统
采用8方向检测算法：
- **四组方向检测**:
  - 水平方向 (左←→右)
  - 垂直方向 (上↑↓下)
  - 主对角线 (左上↖↘右下)
  - 副对角线 (右上↗↙左下)
- **检测逻辑**: 以最新落子为中心，向两个相反方向延伸计数

### 4. 人工智能模块 (DecisionTree)
- **评估函数**: 根据棋盘状态计算得分
- **移动生成**: 优先考虑棋子周围的空位
- **搜索算法**: Alpha-Beta剪枝优化决策树
- **深度控制**: 可配置搜索深度（默认3-4层）

### 5. 棋盘显示系统
- **坐标标记**: 行号(0-14) + 列字母(a-o)
- **棋子符号**: 
  - 黑棋: ● (最新黑棋: ■)
  - 白棋: ○ (最新白棋: □)
  - 空位: ·
  - 中心点: +

## 使用方法

### 软件编译

编译前根据系统修改宏定义
![](image.png)

```bash
mkdir build
cd build
cmake ..
make
```

### 软件运行

在linux系统下

```bash
cd build
./chess.out
```
在windows系统下

```bash
cd build
./chess.exe
```
